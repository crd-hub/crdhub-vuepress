GroupResources represents resource kinds in an API group.

Field Description
group
string 		(Optional)

Group is the name of the API group that contains the resources. The empty string represents the core API group.
resources
[]string 		(Optional)

Resources is a list of resources within the API group. Subresources are matched using a "/" to indicate the subresource. For example, "pods/log" would match request to the log subresource of pods. The top level resource does not match subresources, "pods" doesn't match "pods/log".
