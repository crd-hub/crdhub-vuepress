AutoscalerTrigger (string alias)

(Appears on:ComputeAutoscalerSpec, StorageAutoscalerSpec)

AutoscalerTrigger controls if autoscaler is enabled.

Value Description

"Off"

AutoscalerTriggerOff means the autoscaler is disabled.

"On"

AutoscalerTriggerOn means the autoscaler is enabled.

ContainerControlledValues (string alias)

(Appears on:ComputeAutoscalerSpec)

ContainerControlledValues controls which resource value should be autoscaled.

Value Description

"RequestsAndLimits"

ContainerControlledValuesRequestsAndLimits means resource request and limits are scaled automatically. The limit is scaled proportionally to the request.

"RequestsOnly"

ContainerControlledValuesRequestsOnly means only requested resource is autoscaled.
